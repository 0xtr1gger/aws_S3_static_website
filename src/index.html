Welcome to Just a Page!

This static website has been created as a portfolio project and deployed on an AWS S3 bucket to showcase my skills working with cloud, infrastructure automation, and CI/CD.

Technologies Used:

  • Amazon Web Services (AWS): AWS S3 bucket is used as a cloud-based hosting solution.
  • Terraform: Terraform has been used for the creation of the S3 bucket and deployment of the website as a cloud infrastructure provisioning automation tool.
  • GitHub Actions: GitHub Actions workflow has been set up for the repository to automate CI/CD pipeline responsible for deploying any changes made to the website to this bucket automatically.
  • GitHub Secrets: GitHub Secrets has been used as a secure storage for AWS access keys that Terraform uses to provision the infrastructure.
  • HTML/CSS: This is just how this Just a Page has been written.
